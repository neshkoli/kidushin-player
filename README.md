# שולחן ערוך – הלכות קידושין

נגן שמע סטטי לשיעורי הלכה על הלכות קידושין, נישואים וכתובה מאבן העזר, והלכות עדות מחושן משפט.

## שימוש

פתח את `index.html` ישירות בדפדפן — אין צורך בשרת, תלויות או שלב בנייה.

## מבנה הקבצים

```
index.html      — נגן השמע (HTML + CSS + JS בקובץ אחד)
tracks.json     — מקור האמת לסדר ולקיבוץ הרצועות
*.mp3           — קובצי השמע
```

## הוספת רצועה

1. הכנס את קובץ ה-`.mp3` לתיקייה זו.
2. הוסף ערך לקטע המתאים ב-`tracks.json`:
   ```json
   { "סימן": "כח", "שם": "שם הנושא", "file": "שם הקובץ.mp3" }
   ```
3. שקף את השינוי באובייקט `TRACKS_DATA` ב-`index.html`.

## מבנה tracks.json

```json
{
  "אבן העזר": {
    "הלכות קידושין": [
      { "סימן": "כו", "שם": "במה אישה מתקדשת", "file": "סימן כו אבן העזר.mp3" }
    ]
  },
  "חושן משפט": { ... }
}
```

מפתחות רמה ראשונה: שם הספר. מפתחות רמה שנייה: כותרת הלכות. כל ערך מכיל `סימן`, `שם` ו-`file`.
